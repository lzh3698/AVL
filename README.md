一、项目简介

    C++ AVL树实现，包含核心操作，严格维护AVL树的平衡性质（每个节点的左右子树高度差不超过1）。

二、实现功能

    1、插入操作 (insert)：支持AVL树的标准插入，包含旋转调整（LL、RR、LR、RL）
    2、删除操作 (erase)：支持节点删除，包含删除后的平衡修复
    3、查询操作：
        ① max：获取树中最大元素
        ② min：获取树中最小元素
    4、状态检查：
        ① empty：检查树是否为空
        ② size：获取树中元素个数
    5、自动维护树的平衡

三、算法复杂度

    操作	时间复杂度	空间复杂度
    插入	O(log n)	  O(1)
    删除	O(log n)	  O(1)
    查找	O(log n)	  O(1)
    遍历	  O(n)	    O(1)
    
四、旋转操作

    左旋（LL型）：右子树过高，向右旋转
    右旋（RR型）：左子树过高，向左旋转
    左右旋（LR型）：先左旋后右旋
    右左旋（RL型）：先右旋后左旋

五、项目结构

    AVL_Tree/
    ├── AVL.hpp            // AVL树实现
    ├── main.cpp           // 测试程序
    └── Visual Studio项目文件
    
此AVL树实现确保所有操作后树保持平衡，查询效率稳定在O(log n)。
